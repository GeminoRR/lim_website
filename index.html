Lim Programming Language

The robustness of a strictly typed language, with the simplicity of a dynamic one

main.lim

class user
	let username:str
	let age:int

	func new(_username:str, _age:int)
		username = _username
		age = _age

	func str
		return '{username} is {age}yo'

func main
	let people = new list<user>
	for i from 0 to 5
		people.add(new user("Bob", randint(1, 99)))
		puts(people[-1].str())

Free & Open sources

Lim is based on the principle of open source, meaning that its source code is accessible and modifiable by all. This logical choice encourages collaboration, innovation and transparency in language development. Open source also gives Lim users the opportunity to correct and improve it.

Strictly typed

When the types are not indicated, the compiler takes care of finding them. All this without changing the syntax.

let fibonacci:list<int> = [0, 1, 1, 2]
//Is the same as doing
let fibonacci = [0, 1, 1, 2]

Support for classes

Lim is an object oriented language. It is therefore possible to define these own classes with more or less complexity.

class product
	let name:str
	let id:int

	func new(_name:str, _id = 0)
		name = _name
		id = _id

Designed for minimal runtime

Lim is a programming language designed to simplify the integration of high-level functionality. It uses classes that are equivalent to C structures, without adding complexity. It also features a lazy compiler that compiles only what is necessary.

More examples

Imports

With lim it is easy to import elements from another source file.

//Imports all the public elements of the file "player.lim"
import "player.lim"

//Imports the window library
import window

Exports

With lim it is easy to export variables, functions or classes from a source file. Once our file is imported by another source file, only the elements preceded by the keyword "export" will be accessible (public), the rest will remain inaccessible (private).

//This function is public
export func foo

//This function cannot be used by the file that imported it
func bar

Generic Type Parameters

Currently Lim only supports Generic Type Parameters for classes.
Here is an example of a class that would represent a stack. 

class stack<T>
	let elements:list<T>
	
	func push(elm:T)
		elements.add(elm)

	func remove:T
		return elements.remove(-1)

No need to manage memory

Lim uses a garbage collector, an automatic memory management mechanism that frees memory occupied by unused or inaccessible objects in a computer program. The garbage collector allows developers to concentrate on program logic without worrying about memory allocation and deallocation.

How does it work ?

The Lim programming language has the unique feature of converting its source code into a C file, which can then be compiled into an independent executable file. This allows Lim to be compatible with any operating system and to take advantage of the functionalities of existing C libraries.

For example, Lim can use the SDL2 library to create games or graphics applications. Lim can also optimize the C code it generates using the optimization mode of the chosen C compiler. In this way, Lim guarantees high-performance results.

Execute C code in lim code

Lim is a programming language that lets you integrate C source code into your programs. The syntax allows C code to be inserted at a specific point in the source code and compiled with the rest of the program. This feature can be useful for using libraries written in C, for optimizing certain parts of the code or for taking advantage of features specific to the C language.
